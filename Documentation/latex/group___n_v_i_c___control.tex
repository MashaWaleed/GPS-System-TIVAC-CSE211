This function allows you to set the priority of a specific interrupt in the Nested Vectored Interrupt Controller (NVIC). The priority levels must be within the valid range for your system and should respect the selected PRIORITY\+\_\+\+GROUPING.

\begin{DoxyNote}{Note}
When using PRIORITY\+\_\+\+GROUPING equal to NVIC\+\_\+16\+GROUP\+\_\+0\+SUB or NVIC\+\_\+0\+GROUP\+\_\+16\+SUB\+:
\begin{DoxyItemize}
\item Copy\+\_\+\+Sub\+Priority should be 0.
\item Copy\+\_\+\+Group\+Priority should be in the range \mbox{[}0, 15\mbox{]} for NVIC\+\_\+16\+GROUP\+\_\+0\+SUB and NONE for NVIC\+\_\+0\+GROUP\+\_\+16\+SUB.
\end{DoxyItemize}

When using other PRIORITY\+\_\+\+GROUPING options (NVIC\+\_\+8\+GROUP\+\_\+2\+SUB, NVIC\+\_\+4\+GROUP\+\_\+4\+SUB, or NVIC\+\_\+2\+GROUP\+\_\+8\+SUB)\+:
\begin{DoxyItemize}
\item Copy\+\_\+\+Sub\+Priority can be in the range \mbox{[}0, 3\mbox{]}.
\item Copy\+\_\+\+Group\+Priority can be in the range \mbox{[}0, 7\mbox{]}.
\end{DoxyItemize}
\end{DoxyNote}
