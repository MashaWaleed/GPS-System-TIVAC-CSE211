Specifies the clock source for the Sys\+Tick timer. 

This option determines whether the Sys\+Tick timer uses the processor clock or an external clock source.

\begin{DoxyNote}{Note}
The external clock source must be connected to the STKCLK pin on the microcontroller.

This option is only applicable if the Sys\+Tick timer is configured to use an external clock source (STK\+\_\+\+CLKSOURCE = STK\+\_\+\+CLKSOURCE\+\_\+\+EXT).
\end{DoxyNote}
